3 使用org-mode编辑博客

笔记导出：org-publish

  • 使用 Emacs Org-Mode 写博客 - 日知录
  • 用 Org-Mode 与 GithubPages 搭建个人博客

3.1 导出配置

(setq
      ;; nil as default to avoid leaking sensitive info unintentionally,
      ;; overwritten in per-project settings bellow
      org-html-postamble nil
      org-publish-project-alist
      '(
	("blog-attachments"
	 :base-directory "/mnt/c/myText/local-blog"
	 :base-extension "png\\|jpg"
	 :recursive t
	 :publishing-directory "/mnt/c/myText/blog/images"
	 :publishing-function org-publish-attachment)

	("blog"
	 :base-directory "/mnt/c/myText/local-blog"  
	 :base-extension "org"
	 :recursive t
	 :publishing-directory "/mnt/c/myText/blog/_posts"
	 :publishing-function org-html-publish-to-html
	 :body-only t ;; Only export section between <body> </body>
	 :html-link-use-abs-url t
	 ))

      org-export-allow-bind-keywords t  ; I like "#+BIND:var value"
)
